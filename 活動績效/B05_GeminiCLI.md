## 二、活動目的與宗旨

呼應社團章程第二條，本場活動引入 **Google Cloud GDE 等級** 的講師，讓學員：

1. 認識 Gemini CLI 這款互動式命令列介面 AI 工具
2. 學會利用 LLM 操作本機檔案系統、執行指令、搜尋網路
3. 理解 MCP（Model Context Protocol）標準化規範如何讓 LLM 安全高效地整合外部工具

## 三、活動內容與流程

| 時間 | 內容 |
|------|------|
| 18:45–19:05 | 入場 |
| 19:05–19:15 | 開場 ＆ GDG on Campus TMU 大學介紹 |
| 19:15–20:15 | Gemini CLI 介紹與上手（講師：Max Huang）|
| 20:15–20:35 | 會後交流 / Close |

**核心內容**：

1. **強大操作能力**：實際操作本機檔案系統、執行指令、搜尋網路
2. **終端機指令整合**：使用 `!` 符號執行終端機指令（如 `!ls -l`）
3. **高擴充性**：支援 MCP（Model Context Protocol），讓 LLM 安全高效地整合及存取外部工具和數據
4. **實際應用**：撰寫程式碼、單元測試、IaC（Terraform / Kubernetes）、CI/CD pipeline、Dockerfile、解釋複雜邏輯、分析 log

**特別福利**：Google AI Pro 針對大學生提供免費方案（適用 18 歲以上學生）

## 五、AI 技術應用

本場「主題即 AI」：

- Gemini CLI 本身為 LLM 工具
- MCP（Model Context Protocol）為 AI Agent 的標準化規範
- 學員實作練習資源（Codelabs）由 Google 官方提供
