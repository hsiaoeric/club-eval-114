## 四、Markdown 轉 PDF 的方法

### 方法 1：Pandoc（推薦，最簡單）

```bash
brew install pandoc
brew install --cask basictex   # 中文需要安裝 TeX

cd "評鑑檔本/共通性"
for f in *.md; do
  pandoc "$f" -o "${f%.md}.pdf" --pdf-engine=xelatex \
    -H ../pdf-style-submission.tex \
    -V mainfont="PingFang TC" -V geometry:margin=2cm
done
```

### 方法 2：直接用 Google Docs／Word

1. 把 .md 檔內容複製貼上 Google Docs
2. 整理表格與標題格式
3. File → Download → PDF

### 方法 3：用 VSCode + Markdown PDF 擴充套件

1. 安裝 Markdown PDF 擴充套件
2. 開啟每份 .md
3. 右鍵 → Markdown PDF: Export (pdf)

> **重要**：所有 PDF 內容**不可嵌入外部超連結／影片／其他外部連結資料**（評鑑規定）！文中如有 URL，請改寫為純文字描述。
